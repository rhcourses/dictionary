# Wörterbücher

Dieses Repo enthält Aufgaben, um Structs anhand eines Wörterbuch-Beispiels zu vertiefen.

## Aufgaben

Die Grundlage für das Wörterbuch ist das Package `entry`.
Hier wird ein Struct `Entry` definiert, das einen Eintrag im Wörterbuch repräsentiert.
Dazu gibt es eine Reihe von Methoden, um mit diesen Einträgen zu arbeiten.

Das eigentliche Wörterbuch wird im Package `dictionary` implementiert.
Dabei wird ein Datentyp `Dict` definiert, der eine Sammlung von `Entry`-Objekten verwaltet.
Auch hier gibt es Methoden, um mit dem Wörterbuch umzugehen.

Es empfiehlt sich also, zuerst die Aufgaben im `entry`-Package zu bearbeiten
und danach die Aufgaben im `dictionary`-Package anzugehen.
